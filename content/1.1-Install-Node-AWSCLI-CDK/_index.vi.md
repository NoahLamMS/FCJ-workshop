## 1. Cài đặt Node.js
```bash
# Tải và cài đặt Node.js (phiên bản LTS)
https://nodejs.org/en/download/
```

Kiểm tra cài đặt:
```bash
node -v
npm -v
```

## 2. Cài đặt AWS CLI
```bash
# Tải AWS CLI
https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html
```

Kiểm tra cài đặt:
```bash
aws --version
```

## 3. Cài đặt AWS CDK
```bash
npm install -g aws-cdk
```

Kiểm tra cài đặt:
```bash
cdk --version
```
